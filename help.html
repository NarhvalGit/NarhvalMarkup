Troubleshooting

Common issues and their solutions:

PDF Generation Fails or Times Out

Problem: PDF generation takes too long or fails completely.

Solutions:

  • Try generating with a smaller document first
  • Reduce the number of images or optimize image sizes
  • Close other browser tabs to free up memory
  • Use a different browser (Chrome recommended)
  • Check if your browser is up to date

Markdown Not Rendering Correctly

Problem: Markdown syntax doesn't appear correctly in the preview.

Solutions:

  • Check for proper markdown syntax
  • Ensure there's a space after # symbols in headings
  • Verify list items have proper indentation
  • Check that code blocks have closing backticks
  • Use our sample content as a reference

File Upload Not Working

Problem: Files won't upload or appear in the editor.

Solutions:

  • Ensure the file has a .md or .markdown extension
  • Try using a different upload method
  • Check that the file isn't corrupted
  • Verify the file isn't too large (over 10MB)
  • Try refreshing the page and uploading again

Theme Changes Not Applying

Problem: Theme changes don't appear in the preview.

Solutions:

  • Ensure JavaScript is enabled in your browser
  • Try clicking the theme card again
  • Refresh the page and try again
  • Check if your browser blocks certain scripts
  • Try using a different browser

Frequently Asked Questions

Is my content secure and private?

Absolutely! All processing happens entirely in your browser. Your content never leaves your device, and we don't store or transmit any of your data to external servers. This ensures complete privacy and security.

Can I use this tool offline?

Yes! Once the page is loaded, you can use the converter offline. All the necessary code is loaded into your browser, so you don't need an internet connection to convert markdown to PDF.

What browsers are supported?

We support all modern browsers including Chrome, Firefox, Safari, and Edge. For the best experience, we recommend using the latest version of Chrome or Firefox, as they offer the best performance for PDF generation.

Are there any file size limits?

While there's no strict file size limit, we recommend files under 10MB for optimal performance. Very large documents (over 100 pages) may take longer to process and could potentially cause browser memory issues on older devices.

Can I customize the themes?

Currently, we offer four pre-built themes (Minimal, Academic, Business, and Creative) that are carefully designed for different use cases. Custom theme creation is a feature we're considering for future updates.

Is this tool free to use?

Yes! MarkdownPDF is completely free to use. There are no hidden costs, subscriptions, or limitations. We believe in making professional document creation accessible to everyone.
